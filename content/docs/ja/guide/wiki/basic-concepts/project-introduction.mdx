## 概要

New API は次世代AI基盤プラットフォームであり、AIアプリケーションに統一されたインフラストラクチャを提供します。すべてのAIアプリケーションをホストし、デジタル資産を管理し、未来の統一インターフェースプラットフォームに接続します。

<Callout type="info">
  New API は、OpenAI、Anthropic、Google Gemini、DeepSeek、Midjourney、Suno
  など、30以上のモデルサービスを含む世界の主要なAIサービスプロバイダーとシームレスに統合されています。
</Callout>

### コア特性

- **統一インターフェース**：1つのAPIエンドポイントですべてのAIサービスにアクセスでき、OpenAI標準形式と互換性があります
- **インテリジェントルーティング**：マルチチャネルロードバランシング、自動フェイルオーバー、加重ランダム分散
- **詳細な課金**：回数/量に応じた課金、プリペイドチャージ、複数倍率設定をサポート
- **セキュリティ管理**：トークン権限管理、モデルアクセス制御、API呼び出し監査
- **データインサイト**：リアルタイムダッシュボード、使用量統計、コスト分析
- **マルチテナントアーキテクチャ**：個人開発者、チームコラボレーション、エンタープライズデプロイメントに完璧に対応
