New API オープンソースコミュニティへようこそ！ここでは、他の開発者と経験を共有し、アイデアを交換できます。

## 📜 グループルール

良好なコミュニティの雰囲気を維持するため、以下のルールを遵守してください。

<Callout type="warn" title="重要なお知らせ">
  以下のルールに違反した場合、グループチャットから削除される可能性がありますので、メンバーの皆様はよくお読みになり、遵守してください。
</Callout>

1. 本グループの管理者およびグループオーナーは、皆様に技術サポートを提供する義務を一切負いません。問題がある場合は、[issue](/ja/docs/support/feedback-issues) を提出してください。
2. グループチャットでは、アカウントの売買に関する情報の投稿を禁止します。グループニックネームには、購入誘導や購入に関する情報を表示することを禁止します。
3. 本グループではAPI製品を一切販売していません。いかなる人物（管理者を含む）のAPI製品も信用して購入しないでください。
4. New API は GNU AGPLv3 オープンソースライセンスを採用しており、ライセンスを遵守すれば無料で利用できます。詳細は[プロジェクト紹介](/ja/docs/guide/wiki/basic-concepts/project-introduction)をご覧ください。
5. 本プロジェクトは関連する法令を遵守した上で使用しなければなりません。いかなる違法な話題も議論することを禁止します。

<Callout title="ヒント">
  機能に関する提案やバグのフィードバックがある場合は、GitHubで直接issueを提出することをお勧めします。これにより、問題をより効果的に追跡し、解決できます。
</Callout>
