即夢公式API形式のビデオ生成インターフェースです。

Action パラメーターで操作タイプを指定できます：
- `CVSync2AsyncSubmitTask`: ビデオ生成タスクを送信
- `CVSync2AsyncGetResult`: タスク結果を取得

クエリパラメーターで Action と Version を指定する必要があります。
