現在利用可能なモデルのリストを取得します。

リクエストヘッダーに基づいて、返される形式が自動的に識別されます。
- `x-api-key` および `anthropic-version` ヘッダーが含まれる場合、Anthropic 形式が返されます。
- `x-goog-api-key` ヘッダーまたは `key` クエリパラメーターが含まれる場合、Gemini 形式が返されます。
- その他の場合、OpenAI 形式が返されます。
