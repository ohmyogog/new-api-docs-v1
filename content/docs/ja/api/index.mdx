## 概要

New API は完全な RESTful API インターフェースを提供し、**AIモデルインターフェース** と **管理インターフェース** の2つの主要なカテゴリに分けられます。これらのインターフェースを通じて、AI機能の呼び出しとシステム管理機能を実現できます。

<Callout title="オンラインデバッグ">
  [Apifox 操練場](https://apifox.newapi.ai/) にアクセスしてAPIインターフェースをオンラインでテストおよびデバッグするか、以下のAPIドキュメントを参照してください。
</Callout>
